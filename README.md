# 疯狂猜词 - 亲子互动游戏

一个适合和孩子一起玩的猜词游戏,通过手机重力感应控制游戏进度。

## 🎮 游戏特点

- 📱 手机重力感应控制(向上翻=猜对,向下翻=跳过)
- 📚 520+个适合小学生的日常词汇
- ⏱️ 可选游戏时长(30/60/90秒)
- 🎨 精美的渐变界面设计
- ✨ 流畅的动画效果

## 🚀 如何使用

### 在线访问
直接访问: [游戏链接](https://你的用户名.github.io/crazy-guess/)

### 本地使用
1. 下载所有文件
2. 用手机浏览器打开 `index.html`
3. 授权设备方向权限
4. 开始游戏!

## 📂 文件说明

- `index.html` - 游戏主页面
- `style.css` - 样式文件
- `game.js` - 游戏逻辑
